Project Ideas / Suggestions

  • [Beginner] Contribute to EmojiScreen

    This project was created for people to have a fun and judgement free space to practice contributing on Github. EmojiScreen is a listing of TV shows, movies and musicals depicted with emojis. Add a show or movie or try to add a new feature using JavaScript. If you need help getting started, make an Issue or comment in an exisiting Issue.

  • [Beginner] Random Generator

    Create a web page that generates something randomly at the press of a button. It could be a quote, an image, a color, etc.

    • Random Old Timey Cats

      Example of a random image generator.

    • Random Pantone Colors

      Example of a random color generator.

  • [Beginner] Quiz App

    Create a quiz about your favorite TV show, movie, music artist, random facts, etc.

    • Hamilton Quiz App

      Example of a quiz app.

  • [Intermediate] API Project

    Use an API to display interesting information or allow users to search.

    • What are APIs? (Article)

    • What is an API? (Illustration)

    • Web App Tutorial using the Github API

      Example of a random color generator.

About the Creator

This page includes a number of JavaScript tools, resources and project ideas that I recommend for beginners to review. Originally created for Washington, DC workshops as a resource.

Brittany Walker is a software engineer, instructor and musical theater fan. She is very involved in the DC tech community As a Front End Lead for Women Who Code DC and an Instructor for a free weekly coding workshop for Thinkful.

Brittany also is the co-founder of What the CSS?!, a fun course that teaches CSS via quirky challenges and regularly mentors for those learning how to code.
