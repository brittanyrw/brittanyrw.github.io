Git & Github Setup Resources

  • GitHub

    Create a Github Account. Make sure you remember your password.

  • Download Git and Install Git

  • Download a Text Editor

    This will be used to modify your code. Choose any text editor and open it.

    • Sublime Text

    • Visual Studio Code

  • Open the Command Line

    This is where git commands are run.

    • Open Terminal (Mac)

      Search 'Terminal' using Mac Spotlight Search or go to Applications -> Utilities

      Inside of Terminal, type: git --version command

      Press enter after typing the command. If you have installed Git correctly it will show you the Git version.

    • Git Bash (Windows)

      You must download Git first to get this program. Then search for Git Bash on your computer and open it.
